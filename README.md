
# App de notas

En este proyecto realize una Aplicación web que permite realizar notas personales y guardalas en cada usuario, a partir del registro e inicio de sesión de cada usuario.

## Environment Variables

Esta aplicación funciona con las siguientes variables de entorno

`MONGODB_HOST` this is the Mongodb URI string

`MONGODB_DATABASE` Mongodb database name

`NODE_ENV` node environment



## Running Tests

To run tests, run the following command

```bash
  npm run dev
```
